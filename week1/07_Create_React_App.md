# Create React App

+ [https://facebook.github.io/create-react-app/](https://facebook.github.io/create-react-app/)

```bash
npx create-react-app tic-tac-toe
cd tic-tac-toe
npm start
```

<br>

## npx

+ npm 5.2.0 이상부터 함께 설치된 커맨드라인 명령어

<br>

## npm run build

```bash
npm run build
```
